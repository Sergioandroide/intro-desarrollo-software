CONCEPTO

es una estructura aplicada al desarrollo de un producto de software. Hay varios modelos a seguir para el establecimiento de un proceso para el desarrollo de software, cada uno de los cuales describe un enfoque diferente para diferentes actividades que tienen lugar durante el proceso. Algunos autores consideran un modelo de ciclo de vida un término más general que un determinado proceso para el desarrollo de software

OBJETIVOS DEL CURSO

* Dar a conocer los pasos a tener en cuenta para ingresar al mundo del desarrollo de software.
* Comprender e implementar el concepto OVI como herramienta de información tecnológica. * Conocer las herramientas informáticas de primera mano para la construcción de sitios web de manera práctica y fácil.
